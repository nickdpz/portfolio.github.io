Proyectos

Dashboard Proyecto Mercury

Fuente

Esta página es un tablero de control para un robot. Usando JavaScript envía las instrucciones del teclado y del Joystick virtual usando un Protocolo WebSocket a un servidor. El servidor re direccione las instrucciones a una rasberry pi que tiene implementada el robot. Además, el robot toma imágenes y datos de sensores de posición y los envía para que sean visualizados en el tablero de control. Por otro lado, la página cuenta con una interfaz de autenticación para el usuario pueda entrar al tablero de control. Para el diseño de esta interface se usó JavaScript, JQuery, HTML 5, CSS3 y Framework Bootstrap.
